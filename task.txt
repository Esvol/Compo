ГЛАВНАЯ ИДЕЯ ПРОЕКТА:
Я создаю веб-сервис, предоставляющий платформу для пользователей, 
на котором они смогут размещать информацию о своих айти-проектах. 

1 ЭТАП РАЗРАБОТКИ:
Добавление пользователей(юзеров), которые могут зарегистрироваться на сайт или залогиниться, если уже имеют учетную запись.
Сервис позволит пользователям подробно описать структуру айти-проекта: 
    - название проекта (title) /
    - основную идею своих проектов (idea), /
    - писать некоторый текст для проектов (text), /
    - указать количество участников работающих над проектом (projectTeam), /
    - определить текущий этап разработки проекта (всего будет 5 абстрактных этапов) (stage), /
    - указывать языки программирования на котором пишется проект (через теги можно будет находить эти проекты) (tags). /
    - ставить определенную стоимость проекта (price), /
    - оставлять контактные данные для дальнейшего сотрудничества с компаниями (это будет массив: email + телефон). / 
    - лайкать понравившийся пост (связать с юзером (у юзера будет массив likes)) /
    - добавлять кнопку предзаказа для компании, которые заходят забронировать проект. (такой проект будет показываться как купленный в профиле пользователя, который придумал этот проект).

Пользователю и компаниям можно будет оставлять комментарии под проектами. 
Пользователю и компаниям можно будет также лайкать посты, которые им понравяться. 
Лайкнутые посты будут храниться на страничке (понравившиеся) и фильтроваться с вакансиями.
Компаниям можно будет делать предзаказ по кнопке, оставленной пользователем под постом.

2 ЭТАП РАЗРАБОТКИ:
Компании в свою очередь могут сотрудничать с пользователем по робочему е-мейлу и покупать понравившийся им проект.
Также компании могут выставлять свои посты-объявления для поиска и найма нужного им сотрудника на определенный проект. 
Сервис позволит пользователям подробно описать структуру вакансии (vacancy): 
    - основную идею своих проектов,
    - указать количество участников работающих над проектом,
    - указывать языки программирования на котором пишется проект (через теги можно будет находить эти проекты).
    - ставить определенную ставку для работы "зарплату за месяц", 
    - лайкать понравившийся пост (isLiked - связать с юзером (у юзера будет массив likes))
    - оставлять контактные данные для дальнейшего сотрудничества с пользователями. 

Пользователю можно будет оставлять комментарии под проектами. 
Пользователю можно будет также лайкать посты, которые ему понравяться. 
Лайкнутые посты будут храниться на страничке (понравившиеся) и фильтроваться с айти-проектами.

3 ЭТАП РАЗРАБОТКИ:
Доработать:
    - projectTeam - изменить на объект с количеством учасников проекта и его учасниками с ссылками на их профили.
На веб-сервисе будет доступна премиум подписка, которую можно приобрести по определенной цене. 
Пользователи с премиум подпиской могут пользоваться купленными серверами нашего веб-сервиса 
(что позволит деплоить проект веб-разработчикам), 
а также будут иметь приоритет и показываться другим пользователям и компаниям чаще среди всех остальных проектов.
